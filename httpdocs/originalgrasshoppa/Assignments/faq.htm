How do I Create a Cursor Tail?

First go to http://www.dynamicdrive.com/dynamicindex13/index.html and choose the cursor tail you want. Then copy and paste the html text as described at the site. Viola, you have a cursor tail.The images can be modified by editing the html text but you should be careful in the event that you can mess the coding up.
